ytSlider

ytSlider 一款pc端的滑动插件 提供js版本 （压缩10kb） 和jQuery版本(压缩6kb) 兼容IE6以上浏览器 提供了丰富的参数配置 ，易扩展 。 点击下载

配置参数

参数类型默认值说明
idBox String || Object null 滑动容器节点的id 或 节点的HTMLElement对象（必选）
idScn String null 滑动容器内容节点的id
idNav String null 导航容器节点的id
idText String null 标题容器节点id
idPn String null 左右切换按钮父节点的id
clsScn String slider-scn 滑动内容块节点className
clsText String slider-txt 标题容器节点className
clsNav String slider-txt 导航容器节点className
clsPn String slider-pn 左右切换按钮父节点的className
clsPrev String slider-prev 左右切换左按钮节点的className
clsNext String slider-prev 左右切换右按钮节点的className
clsActive String active 导航按钮当前选中节点的className
clsActive String active 导航按钮当前选中节点的className
eleNav String ol 自动创建的导航节点名称
eleNavChild String li 自动创建的导航节点子元素
activeEvent String mouseover 导航按钮的触发方式 click 或 mouseover
prevText String mouseover 左切换按钮的文本
nextText String mouseover 右切换按钮的文本
isShowText Boolean false 是否显示模块标题
isRunText Boolean true 标题是否动画
isShowPn Boolean true 是否显示左右切换按钮
isShowNav Boolean true 否显示导航按钮
slideType String left 滑动类型 left top fade
slideAuto Boolean true 是否自动滑动
activeIndex Number 0 滑动的开始索引值
speed Number 400 滑动速度（毫秒）
delay Number 5000 滑动延时时间（毫秒）
easing Function || String swing 滑动时的缓动函数
内置 linear、ease、swing、 easeOutQuart
ease 为 js版本才有
设置 ease 后 speed参数无效
initBefore Function null 函数初始化前的回调函数
initAfter Function null 函数初始化后的回调函数
slideBefore Function null 滑动前的回调函数
slideAfter Function null 滑动后的回调函数
